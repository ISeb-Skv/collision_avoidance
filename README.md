# 🚗 Визуальное обнаружение и избегание столкновений беспилотных автомобилей

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![YOLO](https://img.shields.io/badge/YOLOv11n-ultralytics-green.svg)](https://github.com/ultralytics/ultralytics)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Система визуального обнаружения и прогнозирования столкновений для беспилотной мобильной платформы с использованием **YOLOv11n** и **статической камеры**. Робот оснащён тремя цветными светодиодами (зелёный, красный, синий) для определения его центра и ориентации в пространстве.

## 📌 Основные возможности

- **Детекция диодов** с помощью дообученной модели YOLOv11n
- **Вычисление центра и ориентации** робота по трём цветным маркерам
- **Прогнозирование траектории** методом линейной экстраполяции (5 кадров вперёд)
- **Два критерия столкновения**:
  - попадание предсказанного центра в опасную зону
  - превышение порога ширины (расстояния между красным и синим диодами)
- **Три состояния безопасности**: `SAFE`, `WARNING`, `CRITICAL`
- **Визуализация** на кадре: опасная зона, центр, стрелка ориентации, предсказанная траектория
- **Автоматическая остановка** робота при `CRITICAL` (команда на Arduino)

## 🏗️ Архитектура программы
Видео → YOLO детекция → Трекинг (центр + ширина) → Прогноз (5 кадров)
→ Проверка критериев → SAFE / WARNING / CRITICAL → Визуализация

**Аппаратная часть:**
- **Робот:** Arduino Uno, 3 светодиода (зелёный спереди, красный и синий сзади)
- **Камера:** статическая, 30 fps, 1920×1080
- **Вычисления:** Google Colab (GPU Tesla T4) или локальный ПК
